# AGTA OT Connect — Interactive Mockup

A clickable visual concept for **AGTA OT Connect**, a secure web platform connecting
Occupational Therapists with AGTA Home Health Care for equipment recommendations,
referrals, and order management.

**Live preview:** open `index.html` in any browser, or view the deployed link.

## What's inside
- **OT Portal** — AI Equipment Assistant, guided multi-step referral (Client → Clinical →
  Equipment → Documents → Review), My Referrals.
- **AGTA Admin** — dashboard, referral pipeline board, referral detail with documents,
  status, and internal notes.

Use the switch at the top to move between the two views; the sidebar changes screens.

> Visual concept only — sample data, no backend.
